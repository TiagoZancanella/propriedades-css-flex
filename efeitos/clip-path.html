O que é clip path?

O clip-path é uma propriedade que permite criar formas complexas cortando elementos de acordo com um caminho específico. É util para criar layouts e efeitos visuais interessantes.

Como funciona o clip-path

O clip-path define uma área visível de um elemento, escondendo o que está fora dessa área. Existe várias formas de usar o Clip-path Como:

  • polygon(): Define uma forma usando uma lista de coordenadas que formam um polígono.
  • circle():Define uma forma circular.
  • ellipse():Define uma forma elíptica.
  • inset():Define um retângulo interno com uma margem opcional.
