CSS transform scale

O que é scale?

A função scale no CSS é usada para redimencionar elementos, alterado suas larguras e alturas proporcionamente. os valores podem ser:

  • Maior que 1: Aumente o tamanho (ex.: scale: 1.5).
  • Menor que 1: Reduz o tamanho (ex.: scale: 0.5).
